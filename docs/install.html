GDit Installation
Installation
Requirements
  • Windows 10 (Should work Windows 7 or higher)
  • 32-bit
  • At least 2gb of RAM (More highly recommended)
Development version
  1. Download the project
  2. Unzip in location of choice
  3. Navigate to folder in command line
  4. Run ./run.bat c
  5. You're done! :)

Hint: You can open the folder in the command line with Shift + Right-Click
